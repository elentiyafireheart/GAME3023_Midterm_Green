This package is still currently under development.

This Package utilizes Scriptable Objects for the calendar and event systems.

What you'll find:
- A 28-day calendar prefab including a single-day prefab for easy editing.
- Monthly Scriptable Objects that can be updated to accommodate any kind of system.
- Monthly SO include information for the month, used to trigger events.

- Event Scriptable Objects that can be updated to your needs
- Includes information used to trigger events
- can be used to update the calendar by highlighting the numbers in desired colors.
- updates according to events/days passed/current day.
- Premade events that can be updated

- Time system
- Pressing 'Q' triggers opening the calendar in which the game/time will be paused.

- Included snow particle effect
